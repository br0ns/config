1. Install tor and privoxy
2. Add these lines to rc.local:
     /sbin/iptables-restore < PATH_TO_IPTABLES_RULES
     sysctl -w net.ipv6.conf.all.disable_ipv6=1
3. Copy iptables rules (iptables-tor-only) to chosen location
4. Configure browser to use privoxy as HTTP proxy and tor as SOCKS proxy
5. Add these lines to .ssh/config:
     Host *
       ProxyCommand /usr/bin/connect-proxy -S 127.0.0.1:9050 %h %p
6. Don't do stupid things
